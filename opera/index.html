QuasiMod contains custom search engines, ad blocking, buttons and settings for Opera 12.18.

Download QuasiMod for Opera 12.18.
Please read the included text files for more information.

To make installation easier for inexperienced users QuasiMod is now released as a self-extracting 7z that will automatically start the setup program.
The ReadMe/Changelog is shown by the installer before the installation starts (so you can cancel the installation after reading them if you want).
